# /tokenize
Stand up design system foundations.
## Steps
1. **Token architecture** — Define token tiers, naming, and structure using `design-systems` skill.
2. **Color** — Build the palette and semantic color mapping using `ui-design` skill.
3. **Typography** — Create the modular type scale using `ui-design` skill.
4. **Spacing** — Define the base-unit spacing scale using `ui-design` skill.
5. **Adaptive check** — Verify color independence and flexible typography (rem units, scaling, spacing) using `adaptive-interfaces` skill.
## Output
Tokens file (JSON or CSS variables) plus docs covering naming, usage rules, and accessibility notes.
