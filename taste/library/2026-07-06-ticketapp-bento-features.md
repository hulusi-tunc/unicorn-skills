## Analysis
- A features section as a bento grid — uneven tiles (two-up, then two-up, then one wide analytics tile) on a light surface with soft rounding.
- Each tile's asset is a custom mini-scene, not a stock icon: a real product UI fragment (event list, promoter stat card, shareable link bar, guest tiles, bar chart) that demonstrates the feature instead of labelling it.
- The asset tells the story — you understand "Package Selling" or "Trackers" from the illustrated UI, before reading the caption.
- Restrained palette: greyscale base, one desaturated green accent, faded UI screenshots so the composition stays calm.
- Caption pattern: bold short title + one plain-language sentence beneath each tile.

## Steal this
In a bento grid, make each tile a custom mini-scene of the real product doing the thing — the asset demonstrates the feature so the caption barely has to.
