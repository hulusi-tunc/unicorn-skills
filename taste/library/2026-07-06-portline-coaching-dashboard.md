## Analysis
- Layout: calm light-surface SaaS dashboard — left nav, one photo hero banner, then a two-up metric row over a horizontal session carousel. Lots of whitespace, hairline card borders, no heavy shadows.
- Custom data components: not stock charts — a bespoke "Client Transformation Index" ripple/arc gauge and an oversized "87% ↗" momentum stat. Data shown as designed objects, not default widgets.
- Type: huge editorial numerals ("30%", "87%", "14m+") as the visual anchor; quiet grey labels around them. Numbers do the talking.
- Color: near-white base, black text, one warm yellow/amber accent used only on the live data points. Restraint.
- Mood: premium, coaching/executive, confident-calm — the opposite of a dense analytics dashboard.

## Steal this
Design the data component instead of dropping in a chart library — a bespoke gauge plus oversized numerals reads as "premium product," not "admin panel."
